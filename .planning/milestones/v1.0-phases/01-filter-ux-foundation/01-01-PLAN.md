    Hook interface:
    ```typescript
    interface UseDebouncedSearchOptions {
      delay?: number; // default 300ms per D-14
      onChange: (value: string) => void; // parent callback after debounce
    }

    interface UseDebouncedSearchReturn {
      localValue: string;
      setLocalValue: (value: string) => void;
      debouncedOnChange: (value: string) => void;
      cancel: () => void;
      flush: () => void;
    }

    export function useDebouncedSearch(
      initialValue: string,
      options: UseDebouncedSearchOptions
    ): UseDebouncedSearchReturn;
    ```

    Implementation:
    1. Local state (useState) for immediate UI feedback
    2. useDebouncedCallback wrapping parent onChange with 300ms delay
    3. Return both local state setter and debounced callback
    4. Expose cancel/flush for edge cases

    Pattern from RESEARCH.md lines 258-275: immediate local state + debounced parent callback.

    The hook separates immediate visual feedback from delayed expensive operations (URL sync/API calls).
